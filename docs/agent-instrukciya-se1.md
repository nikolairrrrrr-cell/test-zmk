# Как пользоваться агентом — тулза СЕ1 (pdf-zmk2)

В этом чате агент обрабатывает **только** тулзу **СЕ1**: чертежи из папки `СЕ1` → лист Google **«СЕ1»**.

Чтобы агент работал только с СЕ1, в чате можно указать правило: **@.cursor/rules/agent-se1-only.mdc** (или в начале чата написать: «работай только с тулзой СЕ1 по инструкции agent-instrukciya-se1»).

---

## Что делаете вы

1. **Кладёте документы** в папку **`СЕ1`** в корне проекта (изображения чертежей — JPG, PNG или PDF; спецификация стали 09Г2С и т.п.).
2. **Пишете одну команду** в чат — простыми словами (см. ниже).

Терминал не нужен. Запуск CLI, кропы, чтение таблиц и запись в Google Таблицу делает **агент**.

---

## Примеры команд в чат

| Вы пишете | Что сделает агент |
|-----------|--------------------|
| **«Выгрузи СЕ1»** / **«Прогони СЕ1»** | Запустит `pdf-zmk2 full --sheet-write`: кропы по всем файлам из папки СЕ1, при необходимости прочитает кропы и заполнит JSON, затем обновит лист «СЕ1» в Google Таблице. |
| **«Добавил чертежи в СЕ1, выгрузи на лист СЕ1»** | То же: обработать папку СЕ1 и записать результат на лист СЕ1. |
| **«Только проверь СЕ1, в таблицу не пиши»** | Прогон без записи: `pdf-zmk2 full --dry-run` (или с записью отключённой). |
| **«Перезалей СЕ1 заново»** | При необходимости сбросит кэш (удалит старые JSON/кропы по файлам из СЕ1), затем прогонит пайплайн с нуля и обновит лист. |

Достаточно **одной короткой фразы** + файлы уже лежат в **`СЕ1`**.

---

## Что делает агент по шагам

1. Запускает **`pdf-zmk2 full`** (вход — папка `СЕ1`).
2. **Кроп + x4** по каждому изображению → `data/recognition/<slug>.table_x4.jpg`.
3. Если для какого-то кропа ещё нет JSON → CLI вернёт **«прочитать кроп»** (exit 3). Агент **откроет кроп**, снимет таблицу спецификации и запишет `data/recognition/<slug>.json`.
4. Повторно запустит **`pdf-zmk2 full --sheet-write`** → склейка всех JSON и **запись на лист «СЕ1»** в вашей Google Таблице.

Итог: вы только **прикладываете документы в `СЕ1`** и **даёте команду**; агент сам прогоняет тулзу и обновляет лист.

---

## Кратко

**Вы:** файлы в **`СЕ1`** + фраза вроде «Выгрузи СЕ1».  
**Агент:** кропы, JSON по кропам, запуск `pdf-zmk2 full --sheet-write`, лист **«СЕ1»** обновлён.
