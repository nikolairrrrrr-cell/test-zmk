# PDF_ZMK2 — как устроен пайплайн (лист Google **«СЕ1»**)

Терминал вам не обязателен: можно отдать агенту в чат, он запускает CLI и правит JSON.

Пайплайн **из четырёх этапов**, как у База2 (pdf-zmk). **Ни один этап не пропускается.**

1. **CROP + UPSCALE** — CLI по каждому изображению из СЕ1 создаёт `data/recognition/<slug>.table_x4.jpg`.
2. **AGENT READ (гейт)** — CLI проверяет наличие `<slug>.json`; если нет → exit 3 и список «что прочитать».
3. **WRITE JSON** — агент читает кроп, заполняет `<slug>.json`.
4. **NORMALIZE + SHEETS** — повторный запуск: загрузка JSON, склейка, запись на лист Google **«СЕ1»** (переименование: `GOOGLE_SHEET_PDF_ZMK2`).

## Обязательное правило

**Данные для `data/recognition/<slug>.json` берутся только с кропа таблицы** `data/recognition/<slug>.table_x4.jpg`, прочитанного агентом (или человеком) **по самому изображению**.

- **Нельзя** подставлять строки из «эталонных» CSV/JSON или старых чертежей, если вы обрабатываете **новый скан** — таблица на чертеже может отличаться (как у 426-10: листы 212×6, 225×6 и т.д., а не старая спецификация другого варианта).
- Если JSON уже заполнен с кропа, повторный `pdf-zmk2 full` только нормализует и пишет в Google Sheet.

## Что делаете вы

1. Кладёте исходные изображения/PDF в папку **`СЕ1`** (или другую, тогда `--input-dir`).
2. Просите агенту прогнать пайплайн и выгрузить на лист **`СЕ1`**.

## Что делает агент / пайплайн

1. **Кроп + x4** → `data/recognition/<slug>.table_x4.jpg`.
2. **Жёсткий гейт:** пока нет валидного `data/recognition/<slug>.json` с непустым `items`, команда `pdf-zmk2 full` останавливается с **`AGENT_READ_REQUIRED`** (exit 3) — нужно **прочитать кроп** и записать JSON в формате PDF_ZMK2 (`elem`, `det`, `t`, `n`, `section`, `length`, `mass_sht`, `mass_total`, `note`, …).
3. Повторный запуск **`pdf-zmk2 full --sheet-write`** → склейка всех чертежей из папки **`СЕ1`** и запись на **одноимённый лист** в Google Таблице (**`СЕ1`**).

## Кратко

**Вы:** файлы в **`СЕ1`**.  
**Агент:** читает **`*.table_x4.jpg`**, пишет **`*.json`**, затем **`pdf-zmk2 full`** и лист Google.
